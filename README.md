# Juegos-para-Jargim
Para agregar juegos al repositorio, darle click a ad file y luego a "upload files"
luego de terminar la subida dale click a "Commit changes" para completar la subida del archivo

Al final, darle click al archivo jar... luego doble click y copiar enlace donde dice "Raw"
usar ese ultimo link copiado en el sitio de jargimold
